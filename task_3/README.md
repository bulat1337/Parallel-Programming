# Task 3: Circular Message Passing (MPI)

Программа реализует круговую пересылку сообщения типа `int`:

1. Процесс `rank = 0` отправляет начальное значение следующему процессу.
2. Каждый процесс получает значение, увеличивает его на `1`, выводит свой `rank` и текущее значение, затем отправляет дальше.
3. Последний процесс отправляет значение обратно процессу `rank = 0`.
4. Процесс `rank = 0` получает значение, тоже выполняет инкремент и выводит результат.

Таким образом, сообщение проходит полный круг по всем процессам.

## Файлы

- `main.cpp` — исходный код задачи.

## Компиляция

```bash
mpic++ -std=c++17 -O2 main.cpp -o ring_mpi
```

## Локальный запуск

```bash
mpirun -np 4 ./ring_mpi
```

Пример вывода:

```text
Rank 1 received value = 1
Rank 2 received value = 2
Rank 3 received value = 3
Rank 0 received value = 4
```
