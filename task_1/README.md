# MPI Hello World (C++)

Небольшой учебный проект на C++/MPI: каждый процесс выводит сообщение `Hello World`, свой `rank` и общее число процессов.

## Содержимое проекта

- `hello_mpi.cpp` — исходный код программы.

## Требования

- Установленный MPI-стек (например, Open MPI), чтобы были доступны команды:
  - `mpic++`
  - `mpirun`

## Компиляция

```bash
mpic++ -std=c++17 -O2 hello_mpi.cpp -o hello_mpi
```

Что делает команда:
- `mpic++` — компилятор-обертка MPI для C++ (подключает MPI-библиотеки и флаги).
- `-std=c++17` — использовать стандарт C++17.
- `-O2` — оптимизация уровня O2.
- `-o hello_mpi` — имя итогового исполняемого файла.

## Локальный запуск

```bash
mpirun -np 4 ./hello_mpi
```

Что делает команда:
- `mpirun` — запускает MPI-программу.
- `-np 4` — количество процессов (в примере 4).
- `./hello_mpi` — исполняемый файл.

Пример вывода:

```text
Hello World from rank 3 of 4
Hello World from rank 0 of 4
Hello World from rank 2 of 4
Hello World from rank 1 of 4
```

Порядок строк может отличаться от запуска к запуску, это нормально для параллельной программы.
