# Task 2: Harmonic Sum with MPI

Программа считает сумму:

S = 1 + 1/2 + 1/3 + ... + 1/N

`N` передается как аргумент командной строки.
Каждый MPI-процесс считает свою часть суммы,
результат собирается через `MPI_Reduce` и выводится только процессом с `rank = 0`.

## Компиляция

```bash
mpic++ -std=c++17 -O2 main.cpp -o harmonic_mpi
```

## Запуск (локально)

```bash
mpirun -np 4 ./harmonic_mpi 1000000
```

Где:
- `-np 4` — число MPI-процессов.
- `1000000` — значение `N`.
